# proyecto_unida_3

# Sistema de gestión desarrollado con **FastAPI** para que los usuarios puedan administrar su catálogo personal de anime/manga y registrar su progreso de lectura o visualización.

# Tecnologías Utilizadas
* **Framework:** FastAPI v0.110.0
* **Lenguaje:** Python 3.9+
* **Servidor ASGI:** Uvicorn
* **Base de Datos:** SQLAlchemy (SQLite)
* **Documentación:** Swagger UI (OpenAPI 3.1)

# Crear y activar el entorno visual
# python -m venv venv

# Instalar dependencias 
# pip install -r requirements.txt
# pip install pymysql
# pip install fastapi uvicorn sqlalchemy pymysql pydantic

# Iniciar el servidor 
# uvicorn main:app --reload
